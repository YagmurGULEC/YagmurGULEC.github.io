Yagmur Gulec


I have a mechanical engineering background. While studying, I became interested in numerical modeling. So, I did my master in computational fluid dynamics for gasoline injection sprays with Discrete Phase Model in Fluent. During my master journey, thanks to my first coding homework, I became aware of that I am interested in coding. This awareness led me to use a leading open source software so-called OpenFOAM, which is written in C++. Since then, the process of developing programming and computational skills went hand in hand. Since the start of my PhD after obtaining my master degree, I have been a self-taught code enthusiast. However, I thought that it would have been a good idea if I had attended in a professional training. For this reason, I proposed a project for a new hybrid model implementation to carry out in Numap-FOAM summer school in 2018. I developed my programming skills in object-oriented programming in C++. What I know with C++ helped me to write an image processing tool in Java. For data visualization and plotting, I use Python since 2018.
